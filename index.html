JavaScript Game

Guess My Number

To strengthen my grasp of Javascript fundamentals, I followed a video tutorial from a Udemy course and built "Guess My Number," an engaging game where players attempt to guess a hidden number within a limited score (starting at 20, decreasing with each incorrect guess). This project solidified my understanding of core concepts like variables, functions, and if/else statements, while also allowing me to explore additional functionalities:

  • Generating the hidden number: Using Math.random() to generate a random number at the beginning of each game.
  • Dynamic score updates: Employing DOM manipulation to dynamically update the score and high score displays based on user actions.
  • Responding to user input: Implementing event listeners to capture user input, such as clicking a button to submit a guess or selecting the "restart" option.
  • Conditional rendering: Utilizing conditional statements to show or hide elements like the "Game Over" message and "Restart" button based on the game state.

This project not only provided a fun learning experience, but also significantly expanded my Javascript knowledge base.

Dice Game

Expanding my Javascript skills, I followed another Udemy course video and built "Dice Game," a two-player experience. This project solidified my understanding of core concepts (variables, functions, if/else) while exploring additional functionalities:

  • Math.floor() and Math.random(): Generating random dice rolls.
  • Event listeners: Responding to user clicks for rolling and holding.
  • DOM manipulation: Updating player scores and dice visuals.
  • Conditional logic: Implementing turn switching and win conditions.
